# Thumper

Thumper is a 6-wheels unmaned ground vehicle (UGV). 

## Eletronics & Chassis

- Wild Thumper chassis kit (6-wheels 34:1 motors)
- Reconditioned led tape for headlights (any led tape suits)
- Arduino Uno board
- Raspberry Pi 3+ board
- Wires
